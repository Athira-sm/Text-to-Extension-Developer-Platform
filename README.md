# 🚀 Extensify AI 
### AI-Powered Chrome Extension Generator

Extensify AI is a full-stack AI platform that converts natural language prompts into fully functional Chrome extensions. Users can generate, download, and manage extensions without writing any code.

---

## ✨ Features

- 🧠 AI-based code generation (Gemini API)
- 📦 Auto packaging into installable ZIP files
- 📁 Project dashboard (view, edit, download, delete)
- 🔄 Iterative prompt editing (basic version control)
- 🛡️ Code sanitization for security
- 🐳 Dockerized full-stack application

---

## 🏗️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **AI Integration:** Google Gemini API  
- **DevOps:** Docker, Docker Compose  

---

## 📸 How It Works

1. User enters a prompt  
2. AI generates Chrome extension files (JSON format)  
3. Backend converts files → creates ZIP  
4. User downloads and installs extension  
5. Extensions are stored and managed in dashboard
